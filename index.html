Pulumi dev tooling: program testing library for Go

7 December 2023
Built a testing library for running Pulumi programs within a Go test suite. This was primarily to help with testing Pulumi providers but was also adopted by other teams as it's an unopinionated way to test Pulumi programs. It builds on the Pulumi automation API, but providing sensible defaults and asserts for a testing context.

Pulumi AWS VPC component subnet layout improvements

Pulumi CLI: Consume Pulumi programs as components

Pulumi CI Management

Maintained the CI system for over 100 provider repositories for pull-request and release flows.
  • Simplify job matrix for stable job naming and easier tool upgrades.
  • Simplify sentinel check for reliable auto-merge behaviour.
  • Testing release processes via merge-queues & phony provider.

Pulumi GitHub Actions

Wrote a number of small, well defined GitHub actions to help with the publishing of Pulumi providers.
  • pulumi/provider-version-action: Generate the version number for any build. With support for PR builds, pre-releases and tagged releases, and major version upgrade flows
  • pulumi/verify-provider-release: Ensure published Pulumi SDKs & binaries are usable.
  • pulumi/glob-action: Cross-platform glob-based file operations.
  • pulumi/git-status-check-action: Check for unexpected file modifications during CI runs.
